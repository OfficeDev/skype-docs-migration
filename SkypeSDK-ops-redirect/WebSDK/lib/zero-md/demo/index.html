# `<zero-md>` ### Markdown placed inside `<xmp>` tag The markdown that's wrapped inside the `<xmp>` tag is loaded **once** at runtime. If changes to the markdown are made during runtime, manually call the `reload()` method. The *quick* brown **fox** jumps ***over*** the ~~lazy~~ dog. ```html <html> <head> <script src="../../webcomponentsjs/webcomponents-lite.js"></script> <link rel="import" href="../zero-md.html"> <link rel="stylesheet" href="../../prism/themes/prism.css"> <link rel="stylesheet" href="../markdown-themes/default.css"> </head> <body> <zero-md> <div class="md-html"></div> <xmp> Hello World </ xmp> </zero-md> </body> </html> ```
