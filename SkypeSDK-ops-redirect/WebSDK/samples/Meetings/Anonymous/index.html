Permissions

For Skype for Business Online application, you will need the following permission(s):

  • Initiate Real time conversations and join meetings

Provide a conference URI to sign-in anonymously.

After bootstrapping the Skype Web SDK we have access to an api object which exposes an application object. If we create a new instance of an application we get access to a signInManager. The signInManager provides us with a signIn(...) method that we can supply a version, name, and meeting URI to authenticate anonymously. When signing in through this method the application is limited to joining that specific meeting and will not have access to other features such as presence and contact management.
