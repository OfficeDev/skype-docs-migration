Prerequisites

  1. Bootstrap Skype Web SDK
  2. Sign-In

Permissions

For Skype for Business Online application, you will need the following permission(s):

  • Initiate Real time conversations and join meetings

Provide a subject to create a meeting.

The application object exposes a ConversationsManager object which we can use to create new meetings by calling createMeeting(). After creation of the meeting object we can set various properties that will effect the meeting options. In this example we are interested in setting the meeting subject and making the access level available to everyone. We can make these changes via the subject and accessLevel properties. After we have made any changes to our meeting, we need to issue a one-time get() on the onlineMeetingUri property to trigger creation of the meeting on the server side. Once that returns we will have access to the onlineMeetingUri, which we can use to join the meeting via another sample, and joinUrl, which we can provide to Skype for Business desktop client users to join the meeting.
