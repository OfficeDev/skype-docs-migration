Mime

Mime is a simple library that processes messages that adhere to RFC 2046 regarding multipart content type for HTTP requests. Mime parsing is directly used by the Batch library.

Dependencies

  • GeneralHelper.js

Usage

The Mime library is standalone and does not directly require any extra data or objects to construct an instance. The Mime library can take a single optional parameter that will be used as a fall back boundary when parsing the multi-part messages if it is unable to find the boundary in the header. The Mime library processes data via processMessage(...) which takes a single data object with properties headers and responseText. This type of data object is retrieved by using the Transport library's requesting methods. The resulting data object will be an array of familiar objects returned from Transport library requests containing: status, statusText, responseText, results (when responseText is JSON), header, and messageId.
