Tasks

Anonymous Meeting

Anonymous Meeting join will create a UCWA Application and anonymously join a meeting based on supplied data (Conference URI). Once the application is authenticated it is time to wire up event listeners for message (incoming messages), messaging (gather messaging hrefs), and participant (know who is participating in the meeting). After wiring up the event listeners it is possible to start the event channel and make a request to join the online meeting.

The first important event is messaging in a state of Disconnected which will allow making a request to addMessaging which will enable the ability to send messages and leave the meeting. The participant events will allow tracking of who is in the meeting and will be used as a lookup for who recently sent a message to cut down on requests for contact data. The messaging events will contain information about who sent the message and the contents.
